# To-Do List

A simple console-based to do list application written in Python.

## Features
- View the current to-do list
- Add a new task
- Delete a task
- Mark a task as done
- Edit a task
- Toggle showing completed tasks

## How to run
```bash
python todolist.py
```

## Example usage
```
Menu:
1. Show To-do list.
2. Add a new task.
...
Enter your choice: 2
Enter a new task: Buy groceries
Task added!
```
